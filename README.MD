## Getting Started - English Version:

#### 1. Install dependencies

`pip install -r requirements.txt`

#### 2. Copy .env and put values

`cp .env.example .env`

#### 3. Run the following commands in terminal

`$env:FLASK_ENV="development"`
`flask run`
